# 📚 Estudos em Python

Repositório dedicado aos meus estudos em **Python**, contendo exemplos práticos, exercícios, anotações e pequenos projetos desenvolvidos ao longo do aprendizado.

O objetivo é consolidar conhecimentos, testar conceitos na prática e manter um histórico de evolução na linguagem.

---

## 🐍 Conteúdos Abordados

- Fundamentos da linguagem
  - Variáveis e tipos de dados
  - Operadores
  - Estruturas condicionais
  - Laços de repetição
- Estruturas de dados
  - Listas
  - Tuplas
  - Dicionários
  - Conjuntos(Sets)
- Funções e módulos
- Manipulação de arquivos
- Tratamento de exceções
- Programação orientada a objetos (POO)
- Scripts e automações
- Boas práticas e organização de código
